# 组合数生成算法总结

## 一、题目信息

**题目**：从1~n中选出r个数的所有组合
**要求**：按字典序输出所有组合，每个数占3位宽度

## 二、核心思路

### 1. 问题分析

- 组合是无序的，`{1,2}`和`{2,1}`是同一个组合
- 需要输出所有可能的组合，不重不漏
- 输出要求字典序（升序）

### 2. 关键思路

**强制递增法**：规定选择的数字必须严格递增

- 这样每个组合只有一种表示方式
- 自动避免重复（不需要额外去重）
- 天然满足字典序要求

### 3. 算法选择：DFS（深度优先搜索）

- **为什么用DFS**：需要系统地探索所有可能性，DFS能沿着一条路径走到底，然后回溯继续探索
- **本质**：多叉树的深度优先遍历

## 三、代码实现

cpp

```
#include<bits/stdc++.h>
using namespace std;

int r, a[100], n;  // r:要选几个数, a:存组合, n:数字范围

void dfs(int k) {  // 正在确定第k个数
    int i;
    if(k > r) {  // 已经选了r个数
        for(i = 1; i <= r; i++) {
            cout << setw(3) << a[i];  // 场宽3输出
        }
        cout << endl;
        return;
    }
    // 从上一个数+1开始选，保证递增
    for(i = a[k-1] + 1; i <= n; i++) {
        a[k] = i;        // 选择当前数
        dfs(k + 1);      // 确定下一个数
    }
}

int main() {
    cin >> n >> r;
    dfs(1);  // 从第一个数开始选
    return 0;
}
```

## 四、DFS细节解析

### 1. 函数参数设计

cpp

```
void dfs(int k)  // k表示当前正在确定第几个数
```

- `k=1`：选第一个数
- `k=2`：选第二个数
- 以此类推

### 2. 终止条件

cpp

```
if(k > r)  // 已经选了r个数
```

- 当`k=r+1`时，说明前r个数都已确定
- 此时输出当前组合，然后返回（回溯）

### 3. 核心循环

cpp

```
for(i = a[k-1] + 1; i <= n; i++)
```

- `a[k-1]`是上一个已选的数
- `a[0]`初始为0，确保第一个数从1开始
- 这个循环保证了：**每个数都大于前一个数**

### 4. 回溯过程

cpp

```
a[k] = i;    // 选择
dfs(k + 1);  // 递归
// 隐式回溯：函数返回后，自动尝试下一个i
```

## 五、执行过程演示

以n=4, r=2为例：

text

```
开始dfs(1):
├─ i=1 → a[1]=1 → dfs(2)
│  ├─ i=2 → 输出[1,2]
│  ├─ i=3 → 输出[1,3]
│  └─ i=4 → 输出[1,4]
├─ i=2 → a[1]=2 → dfs(2)
│  ├─ i=3 → 输出[2,3]
│  └─ i=4 → 输出[2,4]
└─ i=3 → a[1]=3 → dfs(2)
   └─ i=4 → 输出[3,4]
```

## 六、注意事项

### 1. 数组索引

- `a[0]`作为哨兵，始终为0
- 实际存储从`a[1]`开始
- 输出时也从1循环到r

### 2. 剪枝优化

cpp

```
for(i = a[k-1] + 1; i <= n; i++)
```

- 限制了下界，避免重复
- 实际还隐含上界优化：`i <= n - r + k`（可选）

### 3. 输出格式

- `setw(3)`控制场宽
- 需要包含`<iomanip>`（但`bits/stdc++.h`已包含）

## 七、DFS识别要点

### 什么时候用DFS？

1. **需要探索所有可能性**（组合、排列、子集）
2. **有明确的层次结构**（选第1个、第2个...）
3. **可以剪枝优化**（避免无效搜索）

### DFS解题模板

cpp

```
void dfs(当前状态) {
    if(满足终止条件) {
        处理结果;
        return;
    }
    
    for(所有可能的下一步选择) {
        if(满足剪枝条件) continue;
        
        记录当前选择;
        dfs(下一层状态);
        // 隐式回溯（通过循环尝试下一个选择）
    }
}
```

### 本题DFS特点

- **状态**：当前选到第几个数
- **选择**：下一个可以选什么数
- **终止**：选满了r个数
- **回溯**：函数返回后自动尝试其他值

## 八、时间复杂度

- **时间复杂度**：O(C(n,r) × r)
- **空间复杂度**：O(r)（递归栈深度）

------

**总结**：这是一个经典的DFS应用，通过强制递增避免了组合重复，代码简洁优雅，是回溯算法的典型代表。
