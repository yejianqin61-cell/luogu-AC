# 组合求和为素数问题 - 学习总结

## 📝 问题描述

从 n 个数中选出 k 个数，求所有组合中和为素数的组合数量

------

## 🧠 一、题目思路分析

### 1. **问题拆解**

text

```
整体问题 = 组合问题 + 素数判断问题
         ↓            ↓
    选k个数       判断和是否为素数
```

### 2. **解题流程**

text

```
输入n,k和数组 → 排序数组(可选) → DFS选k个数 → 计算和 → 判断素数 → 计数
```

### 3. **为什么用DFS？**

- 需要**枚举所有组合**（暴力搜索）
- 组合数量不确定，需要**递归探索**
- 有**回溯**需求（选完一个数后要撤销）

------

## 🔍 二、DFS识别条件

### 什么时候用DFS？

| 特征               | 本题表现           |
| ------------------ | ------------------ |
| ✅ 需要枚举所有可能 | 所有k个数的组合    |
| ✅ 每一步选择相似   | 每次选一个数       |
| ✅ 有终止条件       | depth == k         |
| ✅ 需要回溯         | 选完要撤销         |
| ✅ 组合/排列问题    | 组合（顺序不重要） |

### 不适用DFS的情况

- 有最优子结构（用DP）
- 数据量极大（剪枝也超时）
- 有明确数学公式

------

## 📐 三、DFS模板公式

### 基础模板

cpp

```
void dfs(/* 当前状态 */) {
    // 1. 终止条件
    if (/* 达到目标 */) {
        /* 处理结果 */
        return;
    }
    
    // 2. 遍历选择列表
    for (/* 选择 in 选择列表 */) {
        // 3. 做选择
        /* 记录选择 */
        
        // 4. 递归
        dfs(/* 更新状态 */);
        
        // 5. 撤销选择（回溯）
        /* 删除选择 */
    }
}
```

### 本题适配模板

cpp

```
void dfs(int start, int depth) {
    // 1. 终止条件：选够了k个数
    if (depth == k) {
        int sum = accumulate(sel.begin(), sel.end(), 0);
        if (isPrime(sum)) cnt++;
        return;
    }
    
    // 2. 从start开始，避免重复组合
    for (int i = start; i < n; i++) {
        // 3. 选择当前数
        sel.push_back(re[i]);
        
        // 4. 递归选下一个（i+1保证不重复）
        dfs(i + 1, depth + 1);
        
        // 5. 回溯：撤销选择
        sel.pop_back();
    }
}
```

## 💡 四、关键细节

### 1. **组合 vs 排列**

- **组合**：用 `start` 参数，`dfs(i+1)`
- **排列**：用 `visited` 数组，每次从0开始

### 2. **素数判断优化**

cpp

```
// ❌ 慢
for(int i=2; i<sum; i++)

// ✅ 快（只需到平方根）
for(int i=2; i*i <= sum; i++)
```

### 3. **vector使用要点**

cpp

```
// 输入前要分配空间
re.resize(n);

// 排序用迭代器
sort(re.begin(), re.end());

// 添加/删除元素
sel.push_back(x);
sel.pop_back();
```

### 4. **循序渐进法**

text

```
第1步：先写出能选k个数的代码
第2步：加上求和
第3步：加上素数判断

```

### 5. **理解比背模板重要**

模板是拐杖，理解是走路：

- 为什么用start？→ 避免重复组合
- 为什么回溯？→ 恢复状态供下次使用
- 为什么push_back在递归前？→ 记录当前选择
